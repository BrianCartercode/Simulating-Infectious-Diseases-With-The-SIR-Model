This project explores the dynamics of infectious disease spread using the Susceptible-Infected-Recovered (SIR) model and its variants. The model is implemented in Python and used to simulate how changes in transmission rate and intervention timing influence outbreak behaviour.

The project forms part of a research study analysing how mathematical modelling can be used to inform strategies for infectious disease control.

Installation and Usage:

git clone https://github.com/yourusername/sir-model-project.git
cd sir-model-project

pip install numpy scipy matplotlib

python sir_model.py

The initial implementation was adapted from:

https://hackernoon.com/simulating-infectious-disease-spread-with-python-sir-and-seir-models

This project extends the original work by:

Adding multiple infection scenarios
Implementing intervention strategies
Analysing outbreak dynamics
Calculating additional metrics
